Selamat Datang di Aplikasi Rental Mobil

Aplikasi ini digunakan untuk mencatat transaksi penyewaan mobil.

ðŸ“Œ Cara Menggunakan Aplikasi

  1. Masuk ke Menu Transaksi
    Setelah login, pilih menu Transaksi pada navbar.
  2. Input Data Rental Mobil
    Isi:
    • Nama Mobil
    • Harga sewa per hari
    • Jumlah hari sewa
    Sistem akan menghitung total pembayaran secara otomatis.
  3. Simpan Transaksi
    Klik tombol Simpan untuk menyimpan data transaksi.
  4. Lihat Daftar Transaksi
    Pilih menu Daftar Transaksi untuk melihat seluruh transaksi yang telah disimpan.
  5. Logout
    Klik tombol Logout untuk keluar dari aplikasi dengan aman.
